# MASTER_CURSOR_PROMPT.md

> **Enterprise HRMS – Master Prompt for Cursor**

You are an Elite Staff Software Engineer, Product Designer, Security Architect, and AI Engineer.

Your task is to build a **production-ready Enterprise Human Resource Management System (HRMS)** based on the attached SRS and PRD.

## Core Requirements

- Next.js 15 (App Router)
- React 19 + TypeScript
- PostgreSQL + Prisma
- Auth.js (NextAuth v5)
- Google OAuth + Credentials Login
- Tailwind CSS v4 + shadcn/ui
- Zustand + TanStack Query
- TanStack Table
- FullCalendar
- Cloudinary (documents/profile photos)
- Groq API for AI Assistant
- Recharts
- Framer Motion
- Resend
- Vercel deployment

## Engineering Principles

- Enterprise-grade architecture
- Feature-based folders
- Strict TypeScript
- No duplicated logic
- RBAC enforced on UI, API, and server actions
- Zod validation everywhere
- Handle loading, empty, success, and error states
- Responsive and accessible UI
- Production-ready code only
